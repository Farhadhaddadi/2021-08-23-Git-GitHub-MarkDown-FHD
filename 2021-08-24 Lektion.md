# HTML vs Markdown

## Markdown

```Markdown
![tärningar](img/360fx360f.png)
```

#### Resultat

![tärningar](img/360fx360f.png)

### HTML

```html
<img src="360fx360f.png" alt="tärningar"/>
```
#### Resultat

<img src="img/360fx360f.png" alt="tärningar"/>
